# 🐾 PurrCare

**반려묘 건강 관리 웹앱** — 우리 고양이의 식사·배변·약·놀이·수면부터 병원·일정·커뮤니티까지 한 곳에서.

순수 정적 웹앱(빌드 없음)으로, HTML·CSS·바닐라 JavaScript와 `localStorage`만으로 동작합니다.

> 🔗 데모 성격의 개인 프로젝트예요. 서버가 없어 AI 진단·병원 연동·입양 문의·친구 매칭·공유 등은 **로컬 데모**로 동작하고, 데이터는 이 브라우저에만 저장됩니다.

## 🚀 실행 방법

빌드가 필요 없어요. 정적 서버로 열기만 하면 됩니다.

```bash
# 방법 1) 파이썬 내장 서버
python -m http.server 4330
# → 브라우저에서 http://localhost:4330

# 방법 2) 아무 정적 서버나
npx serve .
```

> VS Code라면 Live Server 확장으로 `index.html`을 열어도 됩니다.
> 플로팅 쇼핑 버튼은 별도 배포된 [PurrCare 쇼핑몰](https://slrspdla-nunu.github.io/0610_purrcare/)을 새 탭으로 엽니다.

## 🗂 구조

```
.
├── index.html        # 앱 셸 · 모든 뷰 · 모달
├── style.css         # 디자인 토큰(:root) · 테마 · 컴포넌트
├── script.js         # 상태 · 렌더 · 이벤트 위임 (IIFE)
└── image/            # 앱 이미지
```

> 쇼핑몰은 별도 저장소로 배포되어 있고, 앱에서는 링크로만 연결합니다.

- **테마 시스템**: `:root`의 `--sage` 계열 토큰을 `html[data-theme="…"]`로 재매핑
- **저장**: `localStorage`(`purrcare.cats` / `purrcare.records` / `purrcare.me` / `purrcare.settings`)
